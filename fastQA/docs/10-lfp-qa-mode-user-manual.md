# 磷酸铁锂问答路由手册（现场速用版）

这版只做一件事：**让一线人员在 10 秒内选对路由**。  
不讲空概念，只给“能用”的判断与模板。

---

## 1) 先记住这 4 条（够用了）

- **工艺现场问题先走专利 C**（浆料、涂布、辊压、参数窗口、实施做法）
- **机理解释再补文献 B**（为什么、主导机制、条件依赖）
- **条目/计数题走文献 A**（DOI、有多少篇、哪些文献）
- **侵权/覆盖/规避只走专利 C**（不要用文献单路判断）

## 3) 你关心的示例（明确结论）

问题：`浆料中气泡对浆料粘度的影响`

- **首选：专利 C**
- **次选：文献 B（补机理）**
- 原因：该主题实测命中为 `文献 0 / 专利摘要 1 / 专利分块 180`

---

## 4) 常见误判（会直接误导工作）

- 用文献去回答“是否侵权/是否覆盖” -> 结论不成立  
  - 改：专利 C

- 只看专利实施例就当普适机理 -> 风险高  
  - 改：专利 C 后补文献 B 验证

- 不写 `LFP/磷酸铁锂`，导致检索跑偏到泛材料  
  - 改：问题首句强制加体系限定

---

## 5) 一线可复制提问模板

### A. 工艺现场（默认先专利 C）
`在磷酸铁锂体系中，<工艺问题> 的可实施方案、关键参数窗口、常见失效点分别是什么？`

### B. 机理追问（文献 B）
`针对磷酸铁锂体系，请解释 <现象> 的主导机制，并区分“确定结论”和“条件依赖结论”。`

### C. 覆盖/侵权（专利 C）
`请基于专利库判断 <工艺路线> 与 <专利号/技术主题> 的权利要求覆盖关系与规避点。`

### D. 条目检索（文献 A）
`请列出与 <LFP对象词+工艺词> 相关文献，给 DOI 和关键实验条件。`

---

## 6) 现场执行口令（给测试和工艺同事）

拿到问题先问三句：
- 这是工艺执行问题，还是科学机理问题？
- 我要的是参数方案，还是机制解释？
- 是否涉及权利边界/侵权风险？

对应路由：
- 工艺执行 -> 专利 C
- 机制解释 -> 文献 B
- 条目清单 -> 文献 A
- 侵权/覆盖 -> 专利 C
- 工艺 + 机理 -> 专利 C 后补文献 B
